# Retail Sales Analytics Dashboard

Developed an end-to-end Retail Sales Analytics Dashboard to transform raw sales data into actionable business insights. The dashboard enables KPI tracking, sales performance monitoring, trend analysis, and data-driven decision-making through interactive visualizations and automated reporting.

## Tools & Technologies
- Advanced Microsoft Excel — Data Modeling, Pivot Tables & Power Query
- Dashboard Development & Reporting Automation
- AI-Assisted Analytics (GPT & Claude Sonnet)

- Automated reporting workflows to reduce manual effort
- Analyzed sales trends and business KPIs
- Created executive-level visualizations for decision-making
